# Grocery Backend

Spring Boot backend for a Grocery Management application.

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Gradle

## Features

- Product management
- Category management
- Inventory management
- REST APIs

## Configuration

Configure the following environment variables before running the application:

- DB_URL
- DB_USERNAME
- DB_PASSWORD.

## Port

Runs on port 9000.
